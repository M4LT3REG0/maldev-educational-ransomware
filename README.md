# maldev-educational-ransomware WIP
Laboratorio educativo de ransomware inspirado en Maldev Academy. Implementa cifrado AES‑256 + RSA a nivel de escritorio, footer de cifrado, descifrado con clave privada RSA y una interfaz con cuenta atrás falsa. Incluye build, decryptor y documentación en español para uso exclusivo en entornos de prueba.
